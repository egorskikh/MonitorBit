# MonitorBit

The application allows you to find out the price of bitcoin at the moment and saves the result to the Core Data.

Task: to understand in practice the basics of the CoreDate

Stack:
- MVC
- API
- Parsing JSON
- Core Data
- UIKit
- UIDate
- UITableView
- UITableViewDataSource
- UITableViewDelegate
